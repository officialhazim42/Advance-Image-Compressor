# 🚀 Net Gallery Hb – Advanced Image Compressor
A modern, premium, and fully client-side Image Compressor built using pure **HTML, CSS & JavaScript**.

Compress images instantly in your browser with smart naming, AVIF support, GIF detection, theme toggle, PDF export, and beautiful glass UI.

## ✨ Features

- 🖼 JPG, JPEG, PNG, WEBP Support
- 🆕 AVIF Support (with fallback system)
- 🎞 GIF Upload with Animation Warning
- 🎚 Dynamic Quality Slider (Red → Yellow → Green)
- 📉 Smart Compression Algorithm
- 📊 Professional Size Comparison Chart
- 📄 Convert to PDF
- 🧠 Advanced Smart File Naming System
- 🌙 Dark / Light Theme Toggle (Auto Saved)
- 📤 Floating Share Button
- 💬 Floating Feedback Button
- 🔥 Glass Morphism Modern UI
- ⚡ 100% Client Side (No Server Required)
- 🔒 Privacy Safe (Images never leave your device)

## 🧠 How It Works

1. Upload image
2. Choose format & size
3. Adjust quality slider
4. Click Convert
5. Download compressed file or PDF

All processing happens directly in your browser using Canvas API.

## 🚀 How To Use
